🛍️ E-Shop – Modern E-Commerce Website
A fully responsive E-Commerce Website built using HTML, CSS, Bootstrap 5, and JavaScript.
This project focuses on clean UI/UX, smooth animations, and real-world shopping layout design.

🚀 Live Features
A.🏠 Home Section
1.Full-width Bootstrap Carousel Slider
2.Promotional banners with CTA buttons
3.Smooth transitions and overlay effects

B. 🎯 Offers Section
1.Interactive hover shine animation
2.Multiple discount cards (30%, 50%, 70%)
3.Overlay + call-to-action buttons

C.🛒 Products Section
1.Featured product grid layout
2.Hover image swap effect
3.Product badges (Sale)
4.Star rating system
5.Price with discount (<del> support)
6.Add to Cart button (UI ready)

D.🔥 Special Sale Section
1.Highlight product with description
2.Dynamic countdown timer (JavaScript)
3.Discounted pricing display
4.Responsive layout

E.💬 Testimonials Section
1.Carousel-based testimonials
2.User image + name + company
3.Smooth slider controls

F.📩 Contact Section
1.Contact form (Name, Email, Message)
2.Styled contact info cards with hover effects
3.Clickable:
   📍 Location (Google Maps)
   📞 Phone
   📧 Email
G.Footer Section
1.Multi-column footer layout:
   Brand info
   Quick links
   Support links
   Contact + Social icons
2.Hover animations
3.Developer credit section

🎨 UI/UX Highlights
✅ Modern color system using CSS variables
✅ Smooth hover animations (cards, buttons, images)
✅ Clean typography using Poppins font
✅ Mobile-first responsive design
✅ Consistent spacing & layout system
✅ Shadow & elevation effects for depth
